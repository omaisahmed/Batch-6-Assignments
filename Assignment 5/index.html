What is Comparision Operator?

Comparison Operators are used to test for true or false.


What is IF Statement?

The if(...) statement evaluates a condition in parentheses and, if the result is true, executes a block of code.


What is ELSE IF Statement?

It is a conditional statement performed after an if statement that, if true, performs a function

if (condition is true) { // code is executed } else { // code is executed }
